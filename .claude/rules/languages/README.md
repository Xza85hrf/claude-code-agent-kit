# Language-Specific Rules

Rules in this directory extend the common rules in `../*.md`.

**Priority**: Language-specific overrides common when conflicts exist.

## Usage
Copy the relevant language file to a project's `.claude/rules/` to activate.
Projects using multiple languages can include multiple files.

## Available
- `typescript.md` — TS/JS projects
- `python.md` — Python projects
- Add more: `golang.md`, `rust.md`, `swift.md` as needed
